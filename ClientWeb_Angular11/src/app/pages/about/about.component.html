Get Started

API

  1. Modify the 'Default' connection string in appsettings.json to match your environment
  2. Select the Web project and run 'update-database' command in NuGet Package Manager Console
  3. Run the application by pressing Ctrl + F5

Web app (Angular)

  1. modify the apiURl in 'environment.ts' to match the running API, default is 'https://localhost:5001/api'
  2. Open a terminal in the project folder then run 'ng s -o'
  3. Make sure the api is running with the url that is configured is angular environment

  • Users can select a blog and view the full content (no need to log in at this stage)
